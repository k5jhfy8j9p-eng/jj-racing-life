# JJ Racing Live

Živý program závodního víkendu: https://k5jhfy8j9p-eng.github.io/jj-racing-life/

## Aktualizace harmonogramu

1. Páteční PDF přejmenuj přesně na `patek.pdf`.
2. Sobotní PDF přejmenuj přesně na `sobota.pdf`.
3. V repozitáři zvol **Add file → Upload files** nebo použij odkaz:
   https://github.com/k5jhfy8j9p-eng/jj-racing-life/upload/main
4. Nahraj oba soubory a potvrď **Commit changes**.
5. Po zveřejnění otevři aplikaci, stiskni **Nový víkend** a **Načíst nové PDF**.

Aplikace zkontroluje počet nalezených položek a jízd Junior. Pokud nové PDF nelze přečíst, ponechá poslední funkční rozpis.
